# SmartQuiz

Веб-приложение для студии дизайна интерьера. Проект помогает быстро собрать вводные по будущему интерьеру, показать пользователю ориентировочную стоимость услуг, принять заявку без перезагрузки страницы и передать её в Telegram и в базу данных.

## Что решает проект

SmartQuiz автоматизирует первый контакт с клиентом:

- помогает пользователю пройти визуально понятный квиз по интерьеру;
- собирает тип помещения, зоны, площадь, стиль, бюджет и контакты;
- рассчитывает примерную стоимость на основе данных из базы;
- сохраняет заявки в Django admin;
- отправляет новые заявки в Telegram;
- даёт возможность задать вопросы AI-техподдержке прямо внутри формы.

## Основные возможности

- многошаговый квиз без перезагрузки страницы;
- современный UI с лёгкими анимациями переходов между шагами;
- стили, зоны и помещения подгружаются из моделей Django;
- карточки стилей используют изображения из `StylePictures`;
- AI-техподдержка на базе Gemini;
- Telegram-бот для уведомлений о новых заявках;
- выбор получателя заявок через Django admin;
- готовая Docker-конфигурация для деплоя на сервер.

## Локальный запуск

### 1. Подготовка окружения

Рекомендуемый способ:

```bash
uv sync
```

Если `uv` не установлен:

```bash
pip install uv
uv lock
uv sync
```

### 2. Подготовка переменных окружения

Создайте `.env` на основе [.env.example]

Минимально важные переменные:

```env
SECRET_KEY=change-me
DEBUG=True
GEMINI_API_KEY=your_gemini_api_key_here
GEMINI_SUPPORT_MODEL=gemini-2.5-flash
BOT_TOKEN=your_telegram_bot_token
```

### 3. Миграции

```bash
python manage.py migrate
```

### 4. Запуск веб-приложения

```bash
python manage.py runserver
```

Приложение будет доступно по адресу:

```text
http://127.0.0.1:8000/
```

### 5. Запуск Telegram-бота

Во втором терминале:

```bash
python bot/main.py


Настройка куда отправлять данные о новых пользователях прошедших форму находятся в админ панели
```

### 6. Создание администратора

```bash
python manage.py createsuperuser
```

Админка будет доступна по адресу:

```text
http://127.0.0.1:8000/admin/
```

## Как пользоваться решением

### Сценарий 1. Пользователь оставляет заявку

1. Открывает главную страницу квиза.
2. Проходит шаги формы: помещение, зоны, площадь, стиль, бюджет, контакты.
3. Видит итоговое резюме и ориентировочную стоимость.
4. Отправляет заявку.
5. Заявка сохраняется в базе и уходит в Telegram.

### Сценарий 2. Администратор принимает заявки

1. Заходит в Django admin.
2. Проверяет справочники помещений, зон, стилей и изображений.
3. Выбирает получателя Telegram в разделе настроек бота.
4. Просматривает сохранённые заявки в разделе `Voter`.

### Сценарий 3. Пользователь задаёт вопрос AI-поддержке

Примеры вопросов:

- `От чего зависит цена моего проекта?`
- `Какой стиль лучше выбрать для квартиры 60 м²?`
- `Какие стили наиболее актуальны в форме?`
- `Почему не отправляется форма?`

AI-поддержка отвечает только в рамках вариантов, доступных в текущей форме.
